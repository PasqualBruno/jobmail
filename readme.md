# 🚀 JobMail - Monorepo

Sistema de automação de e-mails para vagas de emprego, desenvolvido com **Node.js/Prisma** no back-end e **React** no front-end.

## 🛠️ Pré-requisitos

- **Node.js** (v20 ou superior)
- **NPM** ou **Yarn**
- Banco de dados **PostgreSQL** (Ex: Supabase)

---

## 🏁 Clonando e Iniciando

### 1. Instalação Automática (Recomendado)

Se você já configurou o script de `setup` na raiz, basta rodar:

```bash
npm run setup
```
